## âš™ï¸ Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/rabiulislam5334/localChef-Bazaar-client.git](https://github.com/rabiulislam5334/localChef-Bazaar-client.git)
    ```

2.  **Install Dependencies:**
    ```bash
    # For Frontend
    cd client && npm install

    # For Backend
    cd server && npm install
    ```

3.  **Environment Variables:**
    Create `.env` files in both folders with your MongoDB URI, Firebase Config, Stripe Keys, and JWT Secret.

4.  **Run the Project:**
    ```bash
    # Client
    npm start

    # Server
    npm run dev
    ```
